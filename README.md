# Custom QR Code Creator

A modern, feature-rich QR code generator with extensive customization options including gradients, logos, frames, and call-to-action text.

## Features

- **Custom Colors & Gradients**: Choose from solid colors, linear, or radial gradients
- **Logo Integration**: Upload custom logos or use the default Lovable heart
- **Multiple QR Styles**: Squares, dots, rounded, and more pattern options
- **Decorative Frames**: Add frames with customizable colors and call-to-action text
- **Smart Color Extraction**: Automatically suggests colors from uploaded logos
- **High-Quality Export**: Download QR codes as PNG images

## Live Demo

**[https://custom-qr-code.lovable.app](https://custom-qr-code.lovable.app)**

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- qr-code-styling

## Development

```sh
# Clone the repository
git clone https://github.com/rainermensing/custom-qr-creator.git

# Navigate to project directory
cd custom-qr-creator

# Install dependencies
npm install

# Start development server
npm run dev
```
